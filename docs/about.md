# About Chitta Doc Hub

Welcome to Chitta Doc Hub — a personal site built with MkDocs Material to showcase documentation, ideas, and blog posts.

This site is designed to help you explore:

- an introduction to the author and the project
- a clean blog menu for different topics
- a modern documentation layout powered by MkDocs Material

## What you'll find here

- **Home**: quick start and core documentation
- **About**: project background and purpose
- **Blogs**: curated articles and notes organized for easy browsing

> This site is built with MkDocs Material, which gives a responsive layout, search, and a polished theme.
