# Writing with MkDocs

MkDocs is a static site generator focused on project documentation. When paired with the Material theme, it becomes an excellent platform for blogs, tutorials, and knowledge bases.

## Why MkDocs

- simple Markdown-based writing
- fast live preview with `mkdocs serve`
- easy navigation and search
- built-in support for a clean site structure

## Getting started

1. Create a new Markdown page in `docs/`
2. Add it to the navigation under `mkdocs.yml`
3. Run `mkdocs serve` to preview locally
4. Publish with `mkdocs gh-deploy` when ready

## Tips

- keep content organized in folders like `blogs/`
- use headings and lists to make articles scannable
- link between pages using relative paths
